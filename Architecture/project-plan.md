# Project Plan

## Project Title
Event Registration & Ticketing System

## Objective
Design and build a serverless event registration system using AWS cloud services.

## Phase 1: Planning & Design
- Define project requirements
- Design system architecture
- Create GitHub repository
- Prepare project documentation

## Phase 2: AWS Infrastructure Setup
- Create DynamoDB table
- Configure IAM permissions
- Create AWS Lambda function
- Configure API Gateway

## Phase 3: Application Development
- Implement registration logic
- Connect Lambda to DynamoDB
- Test API endpoints
- Validate data storage

## Phase 4: Monitoring & Notifications
- Configure CloudWatch logging
- Configure CloudWatch monitoring
- Implement SNS email notifications (optional)

## Phase 5: Testing & Documentation
- Test complete system workflow
- Capture screenshots
- Update GitHub repository
- Prepare final project report

## Expected Outcome
A fully functional serverless Event Registration & Ticketing System built using AWS services.
